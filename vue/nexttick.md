# nexttick

即 下一次 dom 循环更新结束 进行执行 

能够 获取最新 数据

```js
this.$nextTick(() => {
  // 这里的代码会在 DOM 更新完成之后执行
});

//vue3
 nextTick(()=>{
     // 这里的代码会在 DOM 更新完成之后执行
 })
```



## 常见使用场景

1. ✅ **获取更新后的 DOM 元素**
2. ✅ **等视图更新后做动画/滚动操作**
3. ✅ **在单元测试中等待 DOM 渲染完成**
4. ✅ **和第三方插件（如图表、富文本）配合初始化**



## 🧩 总结一句话：

> `$nextTick` 就是告诉 Vue：“你先更新 DOM，等你弄完了，我再做事”。
